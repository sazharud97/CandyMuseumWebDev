MUSEUM OF CANDY

This project was made mostly via HTML, CSS, and bootstrap as part of a Web Development course I took. Javascript was used as well to modify the navbar functionality. --ELABORATION NEEDED--. The first priority was responsive design, which you can see when you shrink or scale the window. I wanted the images and text to be tighter with a larger viewport while still allowing elements some breathing room in smaller viewports.

PALETTE CHOICES

Regarding design aesthetic, I chose to have a slightly darker shade of pink as the default background color. This allowed me to pick paragraph text colors from the image itself that I felt contrasted enough to highlight the section. The same logic applies to headers.

DESIGN PHILOSOPHY

This project was made mostly via HTML, CSS, and bootstrap as part of a Web Development course I took.
